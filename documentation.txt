Лаб раб 3

Цель: закрепление теоретического материала и практическое освоение основных возможностей по реализации алгоритмов и методов обработки изображений.
     
Задача: Написать приложение, реализующее указанные в варианте методы обработки изображений().
       
Реализация: Для начала сделала интерфейс (3 выбадающих списка для выбора цветовой модели, для каждой модели 3-4 поля ввода для точных значений, 
ползунки для каждой модели, текущий цвет и кнопка вызова окна QColorDialog).
            Далее пошла реализация технической составляющей. Использовала QT и C++. Цвета переводятся по формулам по принципу изминили одну 
пересчитали другие. Для удобства перевода созданы глобальные переменные, содержащие значения r,g,b и x,y,z. Добавлено ограничение на ввод (до 
15 символов в строке и 4 знаков после запятой). В конце был создан файл с расширением ехе.

В ходе выполнения лабораторной я создала приложение, позволяющее пользователю выбирать, а затем интерактивно менять цвет, показывая при этом
его составляющие в трех моделях одновременно; закрепила на практике знания, полученные на лекции; напомнила себе работу различных методов и классов Qt.
